# AGENTS.md

## Роль

Ты AI-помощник по разработке SaaS-приложения для управления посуточной арендой жилья.

Твоя задача — помогать разрабатывать проект аккуратно, маленькими шагами, без лишнего усложнения.

## Стек

- Next.js App Router
- TypeScript
- Supabase
- Tailwind CSS
- shadcn/ui
- React Hook Form
- Zod

## Главные правила

1. Не менять архитектуру без объяснения.
2. Не добавлять новые библиотеки без причины.
3. Не реализовывать функции вне MVP без запроса.
4. Не писать огромные файлы.
5. Разделять код по смыслу:
   - app — маршруты;
   - features — пользовательские сценарии;
   - entities — бизнес-сущности;
   - shared — общие компоненты и утилиты;
   - lib — клиенты, конфиги, API.
6. Все формы валидировать через Zod.
7. Все типы описывать явно.
8. Не использовать `any`, если можно избежать.
9. Не хранить секреты в коде.
10. Supabase keys брать только из env.
11. После изменений кратко объяснять:
   - что изменено;
   - какие файлы затронуты;
   - как проверить.

## Архитектурный стиль

Использовать упрощённый FSD-подход:

src/
- app/
- features/
- entities/
- shared/
- lib/

## Поведение

Перед крупной реализацией сначала предложи план.
Если задача маленькая — можно сразу реализовывать.
Не делать оплату, iCal, Авито-интеграции и PWA, пока не будет готов базовый MVP.
