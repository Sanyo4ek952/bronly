# MVP: сервис подборок жилья и календаря занятости

## Цель

Создать SaaS-приложение для людей, которые сдают жильё посуточно или управляют чужими объектами.

Пользователь должен иметь возможность:
- добавлять объекты жилья;
- загружать фотографии;
- отмечать занятые даты;
- собирать подборку объектов для клиента;
- отправлять клиенту публичную ссылку;
- клиент должен открыть ссылку без регистрации и посмотреть варианты.

## Основной пользователь

Собственник, управляющий или посредник, который работает с несколькими объектами и часто отправляет варианты жилья клиентам в WhatsApp/Telegram.

## MVP-функции

### Auth
- регистрация;
- вход;
- выход;
- защита личного кабинета.

### Objects
- список объектов;
- создание объекта;
- редактирование объекта;
- удаление объекта;
- загрузка фото;
- поля:
  - название;
  - описание;
  - цена за сутки;
  - город/район;
  - количество гостей;
  - удобства;
  - статус активности.

### Bookings
- создание брони вручную;
- даты заезда и выезда;
- имя клиента;
- телефон клиента;
- комментарий;
- объект;
- статус брони.

### Calendar
- отображение занятых дат по объектам;
- проверка пересечения дат;
- фильтр по объекту.

### Collections
- создание подборки для клиента;
- выбор дат;
- выбор объектов;
- публичная ссылка;
- страница подборки без авторизации;
- кнопка WhatsApp/Telegram.

## Что НЕ входит в MVP

- онлайн-оплата;
- подписки;
- интеграция с Авито;
- интеграция с Суточно.ру;
- iCal-синхронизация;
- роли сотрудников;
- финансовый учёт;
- мобильное приложение из App Store;
- сложная аналитика.
